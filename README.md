## Getting Started

Install dependencies (use NPM):

```sh
npm install
```

## Run Project

```sh
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Test Project

To create new test, the file must be with `[name].test.js` format.
```sh
# run test
npm test

# check coverage
npm test -- --coverage
```
